# Testing Apple Sign-In on iOS Simulator

## Setup Steps:

1. **Enable Apple ID in Simulator:**
   - Open Simulator
   - Go to Settings > Sign in to your iPhone
   - Sign in with your Apple ID
   - This allows testing Sign in with Apple

2. **Run the app:**
   ```bash
   flutter run
   ```

3. **Limitations:**
   - May not work on all simulator versions
   - Some features might be limited
   - Best on iOS 16.0+ simulators

## Note:
If simulator doesn't work, use Option 2 (Development Mode) below.
